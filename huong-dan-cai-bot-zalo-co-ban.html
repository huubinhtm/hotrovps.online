Cài đặt source Bot và cấu hình hỗ trợ

Tải source Bot Zalo tại: https://huubinhtm.id.vn/products/cmqrnbk5s00mlazhjywsq9mb9

Chuẩn bị các phần mềm dùng để chỉnh sửa code, chạy Bot và hỗ trợ xử lý media.

  • Visual Studio Code: dùng để chỉnh sửa code.
  • Node.js: cài Node.js tầm phiên bản 20 đổ xuống nhé.
  • Xampp: dùng để hỗ trợ môi trường chạy local/database tùy source Bot.
  • Ffmpeg: xử lý các chức năng trong Bot Zalo như !youtube, !tiktok, !zingmp3, …

Hiểu nhanh bộ Ffmpeg

bin/ffmpeg.exe: công cụ xử lý, chuyển đổi video/audio, cắt ghép, nén và đổi định dạng.

bin/ffplay.exe: trình phát video/audio đơn giản.

bin/ffprobe.exe: xem thông tin file media như codec, bitrate, duration, resolution.

Đây là bộ công cụ dòng lệnh để làm việc với video/audio. Ví dụ: đổi .mov sang .mp4, tách âm thanh, nén video, cắt video, v.v.

Tải file hỗ trợ cài đặt

Tất cả file trên mình đã gom hết vào trong đây rồi nhé ae, chỉ việc tải xuống và setup như video mình làm thôi.
